# J2ACoin
## Contenido
En este sprint se crea un documento que habla sobre:
* Nombre de la compañía
* Roles tecnológicos (Backend, Frontend y Testing) 
* Motivación
* Objetivos y subobjetivos
* Casos de uso
* TAM & SAM
* Perfil del cliente.
* Realización de un Business model canvas
* Customer journey
* KPIs & OKR

La idea es orientar el trabajo al proceso de creación de una startup
