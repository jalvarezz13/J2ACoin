## ¿Cómo puedo usar GamesJ3S?

  1. Clona el repositorio con 
```
git clone https://github.com/jalvarezz13/isi-J2ACoin
```
  2. Ejecuta ```yarn start``` en el directorio raíz
  3. Abre en tu navegador: http://localhost:3000/
